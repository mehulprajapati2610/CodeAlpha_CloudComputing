# Cloud Computing Internship Tasks

This repository contains all tasks completed for the CodeAlpha Cloud Computing Internship.

## Tasks Included

1. **Data Redundancy Removal System**
   Prevents duplicate entries by checking incoming data against existing records.

2. **SQL Injection & Data Leak Prevention**
   Secure backend with AES encryption and prepared statements to block SQL injection.

3. **Cloud-Based Bus Pass System**
   Online bus pass generation with cloud-hosted backend and database.

4. **Simple Web-Based Chatbot**
   Rule-based chatbot integrated with a minimal frontend and cloud backend.

## Technologies Used

Node.js, Express, MongoDB Atlas, MySQL, HTML/CSS/JS, React (optional), Netlify, Render, AES-256.

## How to Run

Each task folder has its own README with setup steps.
Most backend tasks run with:

```
npm install
node server.js
```

## Purpose

This repository demonstrates cloud concepts including APIs, security, data handling, and cloud deployment.
